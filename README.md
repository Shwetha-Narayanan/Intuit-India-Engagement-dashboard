## Overview

This project is organized by fiscal year and quarter to facilitate:
- Quarterly performance tracking
- Year-over-year comparisons
- Historical data analysis
- Structured reporting workflows

## Usage

Each quarterly folder contains monthly subfolders for granular data organization:
- **Q1 (Aug-Oct)**: August, September, October data and dashboards
- **Q2 (Nov-Jan)**: November, December, January data and dashboards  
- **Q3 (Feb-Apr)**: February, March, April data and dashboards
- **Q4 (May-Jul)**: May, June, July data and dashboards

Each monthly folder is intended to contain:
- Dashboard files and configurations for that specific month
- Data sources and datasets
- Monthly reports and analytics
- Documentation specific to that month

- **Q3 (Feb-Apr)**: February, March, April

## Getting Started

1. Navigate to the appropriate fiscal year folder (`FY26` or `FY27`)
2. Choose the relevant quarter (`Q1`, `Q2`, `Q3`, or `Q4`)
3. Select the specific month folder for your data
4. Add your dashboard files, data sources, and documentation

## Contributing

When adding new content:
- Place files in the appropriate fiscal year, quarter, and monthly folders
- Include clear documentation for any new dashboards or datasets
- Follow consistent naming conventions across quarters and months
